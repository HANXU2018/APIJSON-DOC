# 文档现状

就开源领域来说， 文档以及网站是开源软件的基础以及核心竞争力，
是使用时非常关注，开发时容易被忽视的领域。
好的文档设计，可以帮助开发者快速地上手以及解决使用问题。
国外已经是比较成熟的行业，IBM、微软、Google等都有相关的职位。
当然国内公司也逐步开始重视起来，菊厂、阿里、腾讯也都开放了相关的职位，并组建了对应的团队。
同时，文档开发是一个典型的应用的领域，
圈子里更多把他理解成技术传播、内容建设，文档只是作为其中之一的载体，可以做的事情其实非常多。
